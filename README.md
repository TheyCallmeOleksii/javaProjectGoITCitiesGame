# Гра "Міста" (Cities Game) 🏙️

Це класична гра "Міста" з графічним інтерфейсом, де гравець змагається проти комп'ютера. Написана на Java 25 з використанням бібліотеки Swing.

## 📜 Правила гри
1. Гравець вводить назву міста.
2. Комп'ютер має відповісти містом, яке починається на останню літеру міста гравця.
3. Винятки: якщо місто закінчується на `ь`, `и`, `й` або `ї`, використовується передостання літера.
4. Міста не можуть повторюватися.
5. Перемагає той, чий суперник не може придумати нове місто або вводить слово `здаюсь`.

## 🚀 Як запустити програму
1. Переконайтеся, що у вас встановлена **Java 25**.
2. Відкрийте проєкт у середовищі розробки (IntelliJ IDEA).
3. Запустіть метод `main` у класі `AppLauncher` (шлях: `src/main/java/AppLauncher.java`).

## ⚙️ Структура проєкту
Проєкт побудований за принципом Separation of Concerns:
* **`game`** — ядро гри: бізнес-логіка (`GameEngine`) та робота з даними (`CityRepository`).
* **`ui`** — графічний інтерфейс: вікна (`WelcomeWindow`, `GameWindow`) та утиліти.
* `AppLauncher` — точка входу (Controller), яка збирає залежності та запускає програму.

## 🗂 Як додати нові міста до бази
База міст зберігається у звичайному текстовому файлі. Щоб додати нові міста:
1. Перейдіть до файлу бази: `src/main/resources/cities.txt`
2. Додайте нове місто з нового рядка.
3. *Регістр не має значення* (програма автоматично форматує текст), але кожне місто має бути на окремому рядку.
